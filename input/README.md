# Input

Dieser Ordner ist nur noch ein Legacy-/Zwischenablagebereich.

Die aktuellen Reifen-Eingaben liegen unter `tyres/input/`. LibreOffice legt hier
noch Lockdateien ab, solange die alten ODS-Dateien offen sind. Nach dem Schließen
von Calc koennen die verbleibenden `TGM Gen V0.33 ... .ods` Dateien nach
`tyres/input/ods/` verschoben werden.
